**题目：** 操作给定的二叉树，将其变换为源二叉树的镜像。

**输入描述：**

```
二叉树的镜像定义：源二叉树 
    	    8
    	   /  \
    	  6   10
    	 / \  / \
    	5  7 9 11
    	镜像二叉树
    	    8
    	   /  \
    	  10   6
    	 / \  / \
    	11 9 7  5
```

**解法：** 

```java
public void Mirror(TreeNode root) {
        if (root==null){
            return;
        }
        TreeNode temp=root.left;//借助中间节点交换树的左右子树
        root.left=root.right;
        root.right=temp;
        if (root.left!=null){
            Mirror(root.left);
        }
        if (root.right!=null){
            Mirror(root.right);
        }
    }
```
