题目内容：

编写程序，把从1到n的n个整数赋予某个int型数组，然后用int型指针输出该数组元素的值。,



输入格式:

输入参数n



输出格式：

输出是1到n n个数组元素,注意最后一个元素之后没有空格。



输入样例：

9



输出样例：

1 2 3 4 5 6 7 8 9

时间限制：500ms内存限制：32000kb
```
#include<iostream>
using namespace std;


int main(){
    int n;
    cin>>n;
    int *p = new int[n];
    for(int i=1;i<=n;i++,p++){
        *p = i;
        if(i==n){
            cout<<*p<<endl;
        }else{
            cout<<*p<<" ";
        }
    }
    return 0;
}
```
