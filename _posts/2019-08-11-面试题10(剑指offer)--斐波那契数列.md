**题目:** 大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n<=39

**解法一(递归)：** 

```java
 private int fibonacci(int n){//
        if (n==0){
            return 0;
        }
        if (n==1){
            return 1;
        }
        return fibonacci(n-1)+fibonacci(n-2);
    }
```

**解法二(动态规划)：** 

```java
private int fibocciCycle(int n){
        if (n==0){
            return 0;
        }
        if (n==1){
            return 1;
        }
        int one=1;
        int two=0;
        int res=0;
        for (int i = 2; i <= n ; i++) {
            res=one+two;
            two=one;
            one=res;
        }
        return res;
    }
```
