**题目：** 输入一棵二叉树，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度。

**解法：** 

```java
public int treeDepth(TreeNode root) {
        if (root == null) {
            return 0;
        }else{//深度为左右子树最深的加上根节点
            return Math.max(treeDepth(root.left) + 1, treeDepth(root.right) + 1);
        }
    }
```
